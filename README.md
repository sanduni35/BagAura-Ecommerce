 **BagAura E-Commerce Web Application**

 Project Description

BagAura is a simple E-Commerce Web Application .
The system allows users to view products and manage a shopping cart. The frontend is built using **HTML, CSS, and JavaScript**, and product data is retrieved from a **Spring Boot REST API** connected to a **MySQL database**.

## Technologies Used

* HTML
* CSS
* JavaScript
* Spring Boot
* Java
* MySQL
* Maven

## Features

* View available products
* Add items to cart
* Remove items from cart
* Update item quantity
* Calculate total price
* Product CRUD operations through REST API

## API Endpoints

| Method | Endpoint           | Description      |
| ------ | ------------------ | ---------------- |
| GET    | /api/products      | Get all products |
| POST   | /api/products      | Add product      |
| PUT    | /api/products/{id} | Update product   |
| DELETE | /api/products/{id} | Delete product   |

## Database

Database name: **bagaura_db**

Note: My MySQL server runs on **port 3307 using XAMPP**.
